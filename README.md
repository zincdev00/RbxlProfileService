## Installation
To install:
1. Install the [dependencies](#dependencies)
2. [Build](#building) the project
3. Import `/build/ProfileService.rbxm` into `ServerStorage/server/packages`

## Dependencies
1. `Class`
2. `Array`
3. `Future`
4. `Parcel`
5. `DataService`

## Building
To build:
1. Run `/build.ps1`
